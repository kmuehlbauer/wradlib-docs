wradlib.trafo.si2kts¶

wradlib.trafo.si2kts(vals)¶

Conversion from SI wind speed units to knots

..versionadded:: 0.6.0

Note

Code was migrated from https://github.com/nguy/PyRadarMet.

Parameters:vals (float) – float or array of floats Speed in SI units (m/s)
Returns:output (float) – float or array of floats Speed in knots

Examples

>>> from wradlib.trafo import si2kts
>>> print(np.round(si2kts(1.), 3))
1.944
