wradlib.georef.polar.sweep_centroids¶

wradlib.georef.polar.sweep_centroids(nrays, rscale, nbins, elangle)¶

Construct sweep centroids native coordinates.

Parameters:
  • nrays (int) – number of rays
  • rscale (float) – length [m] of a range bin
  • nbins (int) – number of range bins
  • elangle (float) – elevation angle [radians]
Returns:

coordinates (3d array) – array of shape (nrays,nbins,3) containing native centroid radar coordinates (slant range, azimuth, elevation)
