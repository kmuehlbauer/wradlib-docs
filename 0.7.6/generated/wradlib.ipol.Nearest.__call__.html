wradlib.ipol.Nearest.__call__¶

Nearest.__call__(vals, maxdist=None)¶

Evaluate interpolator for values given at the source points.

Parameters:
  • vals (ndarray of float, shape (numsourcepoints, ...)) – Values at the source points which to interpolate
  • maxdist (the maximum distance up to which an interpolated values is) – assigned - if maxdist is exceeded, np.nan will be assigned If maxdist==None, values will be assigned everywhere
Returns:

output (ndarray of float with shape (numtargetpoints,...))
